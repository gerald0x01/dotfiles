### How to use?

---

- Install the packages.

- Install the fonts to your font folder. Example: **.fonts** or **/usr/share/fonts/**.

- Update cache fonts.

```shell
$ fc-cache -vf
```

- Replace the i3 config file and polybar files.

- Restart your i3.

- Reboot.
